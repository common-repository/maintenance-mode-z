Maintenance mode for WordPress with many settings

== Description ==

Plugin adds so desired Maintenance mode. And it so easy for you.

- easy settings
- 4 strict and funny templates
- custom image slider
- time based maintenance
- custom logo and custom text
- social icons

For advanced users:

- access rules by IP and user
- custom css

== Installation ==

= Unpack and upload it to the /wp-content/plugins/ directory.

= Activate the plugin through the 'Plugins' menu in WordPress.
